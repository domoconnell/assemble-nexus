# Wallet button assets

These SVGs are interim approximations of the official "Add to Apple Wallet" and
"Add to Google Wallet" buttons so the UI is visually complete while the
wallet integrations are still parked (see
[docs/phases/wallets.md](../../docs/phases/wallets.md)).

When you set up the dev portal accounts and switch the wallet integrations on,
replace these with the **official assets** so brand guidelines are respected
and we get the canonical visual treatment:

- Apple Wallet — download from <https://developer.apple.com/wallet/add-to-apple-wallet-guidelines/>
  (the Marketing Resources zip contains SVG / PDF / PNG versions in every
  approved size and locale).
- Google Wallet — download from <https://developers.google.com/wallet/generic/resources/brand-guidelines>
  (Google provides English + locale variants; the "narrow" style matches the
  height we're using).

Drop the official files in this folder using the same filenames
(`add-to-apple-wallet.svg`, `add-to-google-wallet.svg`) and the ticket card
picks them up automatically.
